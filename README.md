# PrimeEstate — Luxury Real Estate

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

**PrimeEstate** — Elevating the standard of luxury real estate. A full-featured real estate platform with property listings, advanced filtering, property detail pages, and a contact form.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

## Pages

- `/` — Homepage with hero, stats, featured properties, testimonials, and CTA
- `/properties` — Property listings with advanced filters (location, type, bedrooms, price range)
- `/property/[id]` — Individual property detail with gallery, specs, and contact agent form
- `/about` — About page with company story, values, team, and awards
- `/contact` — Contact page with form and office info

## Tech Stack

- **Next.js 14** (App Router)
- **React 18**
- **TypeScript**
- **Tailwind CSS**
- **shadcn/ui** (Radix UI components)
- **React Hook Form + Zod** (form validation)
- **TanStack React Query** (data fetching)
- **Lucide React** (icons)
- **Recharts** (charts)
